我现在需要开发一个星球扩张的小游戏，这个游戏是一个二维游戏，在界面中有多个星球，星球上有文明，分别属于不同派别，
有些星球属于中立派别，中立的星球不会主动去攻击其他星球，星球上的文明每隔一段时间会主动增长，文明的数量决定了星球的强弱
玩家是其中一个派别，玩家需要控制星球之间的连线去和不同星球之间建立连接，然后通过连接，星球上的文明可以去攻击其他星球，
建立连接需要消耗星球上的文明，如果距离过远，则无法建立连接，建立连接后，星球会派出文明去攻击其他星球，派出的文明不会消耗星球上的文明，
攻击的结果是，被攻击的星球上的文明会减少，如果被攻击的星球上的文明数量为0，则该星球会被占领，
可以手动端开己方的连接，如果对方也连接了己方，则断开连接时己方的连接会回到己方的星球，并恢复创建连接时消耗的文明数量，
且对方的连接会延长到己方的星球，如果对方的文明数量不够延续连接需要消耗的数量则会自动收回连接并恢复连接消耗的文明数量，
如果对方没有和己方连接则断开连接时，从断开位置分为两部分，一部分回到己方的星球，并恢复创建创建这部分连接时消耗的文明数量，
另一部分会去到对方星球，并根据创建这部分连接时消耗的文明数量去抵消对方星球的文明数量，如果对方星球的文明数量不足，则会被占领，
如果连接的时己方的星球，则派出的文明会增加己方星球的文明数，
